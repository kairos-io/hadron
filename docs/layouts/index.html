Hadron Linux

Minimal. Trusted.
Built for the cloud & the edge.

Hadron is a minimal, upstream-aligned Linux distribution built from scratch using musl, systemd, and vanilla Linux kernels — engineered for secure, predictable cloud and edge deployments.

Minimal & Lean
Bare essentials only — no downstream patches, no vendor ecosystem.
Trusted by default
Secure-boot ready, USI/UKI supported as first-class citizens.
Expandable
Build your own or use sysextensions to expand the system easily.
Safe Atomic Updates
A/B upgrades with automatic failovers, minimal failure surface.
Simplify configuration
Configure your nodes with simple and declarative YAML files.
Trusted Supply Chain
Secure and measured boot with full data partition encryption.
Hadron + Kairos

The purpose-built stack for immutable, reliable operations.

Kairos provides the image-based lifecycle, A/B upgrades, configuration engine, and secure supply chain that unleash Hadron's full potential. Together, they form a minimal, predictable, and production-ready foundation for cloud and edge systems.
