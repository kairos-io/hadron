HADRON by Kairos

The foundation for image-based systems.

Hadron is a minimal, upstream-aligned Linux distribution built from scratch with musl, systemd, and vanilla kernels — designed for predictable cloud and edge environments.

It ships without a package manager by design. Build your image, deploy it, and let Kairos handle lifecycle management.

Engineered by the Kairos team at Spectro Cloud.
